## Why Autotel?

<CardGrid>
  <Card title="One Import" icon="rocket">
    `init()`, `trace()`, `getRequestLogger()`, `createStructuredError()` — everything from `autotel`.
  </Card>
  <Card title="Trace Any Function" icon="seti:typescript">
    Wrap handlers with `trace()` and get automatic spans, error recording, and context propagation.
  </Card>
  <Card title="Structured Errors" icon="error">
    Throw `createStructuredError({ message, why, fix, link })` — clients parse with `parseError()`.
  </Card>
  <Card title="Product Events" icon="list-format">
    `track('event.name', attrs)` fans out to PostHog, webhooks, or any subscriber.
  </Card>
</CardGrid>

## Quick Start

```bash
npm install autotel
```

```typescript
import { init, trace, getRequestLogger } from 'autotel';

init({ service: 'my-api', endpoint: 'http://localhost:4318' });

export const postCheckout = trace((ctx) => async (req, res) => {
  const log = getRequestLogger(ctx);
  log.set({ userId: req.user.id });

  const result = await processCheckout(req.body);
  log.set({ orderId: result.id });
  log.emitNow();

  return res.json(result);
});
```
